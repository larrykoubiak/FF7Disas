  • Opcode: 0xF8
  • Short name: PMVIE
  • Long name: Set Movie

Contents

  • 1 Memory layout
  • 2 Arguments
  • 3 Description
  • 4 FF7 Movie ID List

Memory layout

0xF8 M

Arguments

  • const UByte M: Movie ID to set.

Description

Sets the movie file that will be played at a future point in the script by MOVIE.
